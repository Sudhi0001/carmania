# Carmania 🚗

A premium car customization platform built using React that allows users to explore, customize, and save car builds with real-time pricing.

---

## 🌐 Live Demo

(https://carmania-omega.vercel.app/)

---

## 🚀 Features

* Browse curated car builds
* Interactive Dashboard (Garage)
* Save and manage custom builds
* Advanced Car Configurator:

  * 🎨 Color selection (visual swatches)
  * 🛞 Wheels customization
  * 🪑 Interior options
* 💰 Real-time price calculation
* 📩 Contact dealer with full configuration details
* ⚡ Smooth UI interactions and transitions

---

## 🧠 What Makes It Unique

Carmania includes a **dynamic car configuration system**, similar to real-world platforms like Tesla, where users can customize vehicles and instantly see pricing updates.

Unlike basic CRUD apps, this project focuses on:

* Complete user flow (Explore → Customize → Save → Contact)
* Interactive UI behavior
* Product-level thinking
